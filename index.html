プロパティまとめ

HTML / CSS / JavaScript プロパティ集

HTMLとは

HTMLとは「Hyper Text Markup Language」の略であり、Webサイトの構造を作る言語である。
<a></a>アンカーリンク
<button></button>ボタン
<h1></h1>見出し
などがある。

CSSとは

CSSとは「Cascade Style Sheet」の略であり、Webサイトの見た目を整える言語である。
font-size/文字サイズ指定
margin/外側余白
padding/内側余白
background-color/背景色
などがある。

JavaScriptとは

HTMLとCSSとJavaScriptは相対関係にあり、しばしば「人体や車」に例えられる。
HTMLが骨格を担い、CSSが肉を担うとするならJavaScriptは神経や伝達信号を担う。Webサイトに命を吹き込む言語である。
