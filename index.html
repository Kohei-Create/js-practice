Mathオブジェクト（数学関数）

高度な計算を行うには、Mathオブジェクトの数学関数を使います。記述するときは、
new演算子でオブジェクトの宣言をしません。次のように、すぐプロパティやメソッドを付けて利用します。

働き 使い方 計算結果
PI 円周率 a = Math.PI; 約3.141592653589793
プロパティ
abs（） 絶対値 a = Math.abs（-100）; 100
ceil（） 切り上げ a = Math.ceil（100.4）; 101
floor（） 切り捨て a = Math.floor（100.4）; 100
max（） 最大値を返す a = Math.max（1, 8, 3）; 8
min（） 最小値を返す a = Math.min（1, 8, 3）; 1
pow（） べき乗 a = Math.pow（4,2）; 16（4の2乗）
random（） 乱数を発生させる a = Math.random（）; 次項参照
round（） 四捨五入 a = Math.round（100.4）; 100
sqrt（） 平方根 a = Math.sqrt（25）; 5
